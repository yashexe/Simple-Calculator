# Simple Calculator app

Crunch numbers with the Simple Calculator [here!](https://yashexe.guthub.io/Simple-Calculator)

Contains functionaility for:
- Addition
- Subtraction
- Multiplication
- Division
- Powers
- Log/Ln/e
- Parentheses
- All clear (AC)
- Delete 
- sin/cos/tan
- Answer
- Return Answer
